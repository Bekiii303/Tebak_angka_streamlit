# 🎮 Tebak Angka (Streamlit App)

Sebuah game sederhana berbasis Python dan Streamlit di mana pemain mencoba menebak angka acak antara 1 hingga 100.

Project ini dibuat sebagai latihan logika pemrograman, state management (`session_state`), dan interaksi user menggunakan Streamlit.

## 🎯 Cara Bermain

- Masukkan angka tebakan (1–100)
- Klik tombol **"Tebak"**
- Sistem akan memberi petunjuk:
  - 📉 Terlalu kecil
  - 📈 Terlalu besar
  - 🎉 Benar
- Lihat jumlah percobaan kamu
- Gunakan tombol **"Reset game"** untuk mulai ulang

---

## ✨ Fitur

- Angka acak setiap game baru
- Sistem percobaan (attempt counter)
- Feedback interaktif (info, warning, success)
- Reset game untuk memulai ulang permainan
- UI sederhana dan mudah digunakan
